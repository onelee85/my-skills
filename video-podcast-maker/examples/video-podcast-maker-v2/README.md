# Video Podcast Maker V2 - Example Project

This is a complete example project generated by the [Video Podcast Maker](https://github.com/Agents365-ai/video-podcast-maker) Claude Code skill.

## Project Structure

```
video-podcast-maker-v2/
├── topic_definition.md          # Step 0: Topic planning
├── podcast.txt                  # Step 3: Narration script with section markers
├── timing.json                  # Step 7: TTS timing data

```

## How to Use This Example

### 1. View the Script
Check `podcast.txt` to see how section markers `[SECTION:xxx]` structure the content.

### 2. Study the Timing
`timing.json` shows how TTS generates frame-accurate timing for each section.

### 3. Explore the Component
`VideoPodcastMakerV2Video.tsx` demonstrates:
- Section-based video composition
- Animation patterns (FadeIn, SpringPop)
- Chapter progress bar integration
- 4K rendering with scale(2) technique

### 4. Reproduce
To create your own video with this skill:
```bash
# In Claude Code
/video-podcast-maker
```
